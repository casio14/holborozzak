# Esemény-források

Az `/esemeny-gyujtes` skill ezt a listát járja végig: kigyűjti a borhoz köthető
eseményeket, és az újakat jelöltként beküldi az admin → Jelöltek oldalra.

Egy sor = egy forrás. A `-` utáni URL a kiindulópont (lista-/programoldal);
megjegyzést a `—` után írhatsz (pl. mire figyeljen az adott oldalon).
